8. Create Second Table for Join (Requirement c)
CREATE TABLE airports (
    name STRING,
    country STRING,
    area_code INT,
    code STRING
)
ROW FORMAT DELIMITED
FIELDS TERMINATED BY ',';


9. Load Airports Data
LOAD DATA LOCAL INPATH '/mnt/airports.csv'
INTO TABLE airports;


10. Join Tables (Requirement c)
SELECT a.name, f.flight, f.dep_delay
FROM flight_data f
JOIN airports a
ON (f.origin = a.code);


11. Create Index (Requirement d)
CREATE INDEX flight_index
ON TABLE flight_data (origin)
AS 'COMPACT'
WITH DEFERRED REBUILD;
ALTER INDEX flight_index ON flight_data REBUILD;
SHOW INDEX ON flight_data;


12. Average Departure Delay per Day in 2008 (Requirement e ✅ IMPORTANT)
SELECT day, AVG(dep_delay) AS avg_delay
FROM flight_data
WHERE year = 2008
GROUP BY day;

# Important Definitions

## Database

* Collection of related data stored electronically.

## Table

* Structured format containing rows and columns.

## External Table

* Hive table where data remains outside Hive warehouse.

## Join

* Combines data from multiple tables.

## Index

* Improves query search speed.

## Aggregate Function

* Function performing calculations on multiple rows (AVG, SUM, COUNT).

## 20 Important Viva Questions and Answers

### Hive Flight Database Practical

---

### 1. What is Apache Hive?

* Apache Hive is a data warehouse tool built on Hadoop used for querying and analyzing large datasets using SQL-like language.

---

### 2. What is Hadoop?

* Apache Hadoop is a framework used for distributed storage and processing of big data.

---

### 3. What is HDFS?

* HDFS (Hadoop Distributed File System) is the storage system used in Hadoop.

---

### 4. What is a Database in Hive?

* A database is a collection of related tables stored together.

---

### 5. What is the use of `CREATE DATABASE`?

* It creates a new database in Hive.

---

### 6. Why do we use `USE database_name`?

* It selects the database for current operations.

---

### 7. What is a Table in Hive?

* A table stores data in rows and columns.

---

### 8. What is the difference between Managed Table and External Table?

* Managed Table: Hive controls both metadata and data.
* External Table: Hive controls only metadata; data remains in HDFS location.

---

### 9. What is the use of `DROP TABLE`?

* It deletes a table from the database.

---

### 10. What is `ALTER TABLE` used for?

* It modifies the structure of an existing table.

---

### 11. What is the use of `LOAD DATA` command?

* It loads data from local filesystem or HDFS into Hive table.

---

### 12. What is `ROW FORMAT DELIMITED`?

* It specifies that table data is separated using delimiters like commas.

---

### 13. What does `FIELDS TERMINATED BY ','` mean?

* It indicates that columns are separated using commas in CSV file.

---

### 14. What is the purpose of `INSERT INTO`?

* It adds records into a table.

---

### 15. What is a JOIN in Hive?

* JOIN combines rows from two or more tables based on a common column.

---

### 16. What is an Alias in SQL/Hive?

* An alias is a temporary short name given to a table or column.

---

### 17. What is an Index in Hive?

* An index improves query searching speed on specific columns.

---

### 18. What is the purpose of `GROUP BY`?

* `GROUP BY` groups rows having same values for aggregation.

---

### 19. What does the `AVG()` function do?

* `AVG()` calculates the average of numeric values.

---

### 20. Why is Hive used in Big Data?

* Hive simplifies big data analysis using SQL-like queries on Hadoop datasets.
