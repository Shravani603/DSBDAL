# Important Definitions

## HiveQL

* SQL-like query language used in Hive.

## External Table

* Table where data remains outside Hive warehouse.

## Join

* Combines rows from multiple tables.

## Index

* Improves query performance.

## Aggregate Function

* Function performing calculations on multiple rows like AVG, SUM, COUNT.







































# 20 Important Viva Questions with Short Answers (Hive – FlightDB)

---

## 1. What is Hive?

**Answer:**
Apache Hive is a data warehouse tool used on Hadoop for querying and analyzing large datasets using HiveQL.

---

## 2. What is HiveQL?

**Answer:**
HiveQL is an SQL-like language used in Hive to perform database operations.

---

## 3. What is the difference between Hive and SQL?

**Answer:**
SQL works on traditional databases, while Hive works on Hadoop distributed storage.

---

## 4. What is a database in Hive?

**Answer:**
A database is a collection of related tables.

---

## 5. Why do we use `CREATE DATABASE`?

**Answer:**
It is used to create a new database in Hive.

---

## 6. What is the purpose of `USE FlightDB`?

**Answer:**
It selects the FlightDB database for performing operations.

---

## 7. Why is `DROP TABLE IF EXISTS` used?

**Answer:**
It deletes a table safely without showing an error if the table does not exist.

---

## 8. What is the difference between Managed Table and External Table?

**Answer:**

* Managed Table → Hive controls data and metadata.
* External Table → Hive controls only metadata; data remains in HDFS.

---

## 9. What is HDFS?

**Answer:**
Hadoop Distributed File System is Hadoop’s distributed storage system used to store big data files.

---

## 10. What does `ROW FORMAT DELIMITED` mean?

**Answer:**
It tells Hive that the data fields are separated using delimiters like commas.

---

## 11. Why do we use `FIELDS TERMINATED BY ','`?

**Answer:**
It specifies that columns in the file are separated by commas.

---

## 12. What is the use of `STORED AS TEXTFILE`?

**Answer:**
It stores table data in plain text format.

---

## 13. Why was `INT` changed to `STRING` in the table?

**Answer:**
To avoid NULL values caused by invalid numeric data.

---

## 14. What is the purpose of `LOAD DATA` command?

**Answer:**
It loads data from local system or HDFS into a Hive table.

---

## 15. Difference between `LOCAL INPATH` and `INPATH`?

**Answer:**

* `LOCAL INPATH` → Loads file from local Linux system.
* `INPATH` → Loads file from HDFS.

---

## 16. What is a JOIN in Hive?

**Answer:**
JOIN combines records from multiple tables based on a common column.

---

## 17. Why is `GROUP BY` used?

**Answer:**
It groups similar records together for aggregate operations like AVG, SUM, COUNT.

---

## 18. What is the use of `AVG()` function?

**Answer:**
It calculates the average value of a numeric column.

---

## 19. What is indexing in Hive?

**Answer:**
Indexing improves query searching speed on specific columns.

---

## 20. What is the purpose of `WHERE` clause?

**Answer:**
It filters records based on a condition.
